What is local share?
Local Share is an data sharing software within a LAN(Local Area Network) supporting android, Linux and windows platforms.
While in sending mode, my friend was not visible on the list?
Even if you are sending over a LAN and your friend is not visible it can be either due:
1. Your friend may be receiving using his own hotspot in android phone.
2. You and your friend is not on the same subnet. But you can still share data by his/her IP address which is displayed on home screen. Click on 'send using IP' button and enter your friend's ip address.
Why is 'send to android' button on Desktop application?
When you connect your PC to the android hotspot which is active for recieving file, the device will not be there in online users list. But you can send the file by clicking send to android button.
Does this work only over Local Area Network?
Along with Local Area Network, you can also use your own hotspot for high speed data transfer.
Is sharing using own hotspot was not sufficient?
Sharing using own network do provide high speed but it give less freedom of distance. In today's world most of the institutions and offices are planted with Local Network Connectivity which connect them always. This application targets to those type of users also.
Can I share over different subnets?
If the subnets are part of the same Local Network then you can send by 'Send using IP' button.
What is jar distribution in downloads ?
To run jar file, you must have java run time environment (jre or jdk) intalled on your PC and and must support JavaFx. This distribution needs no installation and it is of very less size.
